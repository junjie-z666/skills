# ZJJ Skills

## 安装

```bash
npx skills@latest add zjj/skills
```

## 技能列表

| 技能 | 功能                                                                     | 使用与效果 |
|------|------------------------------------------------------------------------|------------|
| `maestro` | 通过 Maestro MCP 自动操作手机，验证移动端 app 开发/修复结果，内置一些避坑规则| 开发或修复功能后，AI 自动构建部署 app 到模拟器，通过 Maestro MCP 操作手机导航到目标页面，逐步验证功能是否符合预期，实现"开发→部署→验证"闭环 |
